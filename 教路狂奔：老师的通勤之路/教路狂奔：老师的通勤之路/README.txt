本游戏——教路狂奔：老师的通勤之路是饭寒暄粉丝会的训练项目

本游戏背景为老师在两个校区通勤

点击Game.exe开始游戏，如没有账号需先注册账号，有账号可登陆。账号密码输入应为1-10位的字符串，由数字或大小写字母组成，中间不含空格。

进入游戏可选择关卡或购买载具，初始开放第一关和海陆空各一个载具。1、4、7关为陆地关卡，2、5、8关为空中关卡，3、6、9关为海上关卡，难度均依次上升，可以通过在关卡中获得零件来开发新载具，金零件相当于三个、银零件相当于两个、铜零件相当于一个。

在游戏中碰到障碍物会减少油量，碰到油箱可以增加油量，油量为0就失败，每一关路程达到一千就过关，即老师到达。只有通过前一关才能解锁下一关。

在陆地游戏关卡，用户可通过点击上控制汽车跳跃躲避障碍物，踩障碍物上方可以二次跳跃。
在空中游戏关卡，用户可通过点击上控制飞机起飞，在起飞后会有短暂的跳跃然后回向下掉落，飞机驶出界面外游戏失败。
海上游戏关卡，用户可通过上下控制船移动，船可以获得本泳道和上一泳道的油箱。
